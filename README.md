# AHK Auto Clicker
* Free and Simple
* Written by AutoHotkey
## Usage
* F2: Add to playlist.
* F3: Play.
* F4: Stop.
* Right-Click menu: clear and clear all items in playlist; save and open playlist.
* Suport drop files.
